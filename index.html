The VISAB Toolset

Welcome to the VISAB online documentation powered by DocFX!

VISAB stands for VISualizing Agent Behaviour and can be bestly described as a tool that is capable of communicating with games to extract game information und illustrate it for AI developers to have a perfectly tailored insight of the bot implementation's behaviour with perfectly tailored views for each specific game. To cover the full stack of required use cases VISAB comprises a toolset which contains the following:

  • VISAB: A standalone maven-powered Java application, usable on Windows, Mac and Linux that provides a variety of features for AI behaviour debugging.
  • VISABConnector: A C# library that can be deployed in a unity game to communicate with VISAB and provide the necessary information for visualization.
  • VISABConnector.Unity: A kind of sub module within the VISABConnector that can be used to perform snapshots of real game objects and send them to VISAB to enhance the user-experience by having the exact same visuals and map just as in the original game.

Online Documentation Content

This online documentation aims at helping further developers to understand how VISAB and its toolset works and guide further extensions to it. Under Documentation you will find more general information on how VISAB is built, how it is working and what the underlying concepts are. Getting Started aims at helping both users and developers to get a starting point for the VISAB toolset and by exposing hands-on guides on various things. In Code Samples some abstracted sample implementations with the example of Tetris are shown to give a deeper insight on how to extend the code base to support further games. Furthermore the API code documentation of the VISABConnector and VISAB can also be viewed.
